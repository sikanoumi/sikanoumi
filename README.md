# 🌱 Yomi（ヨミ） | 感情×世界観の生成AIエンジニア

創作キャラクターの「感情・人格・成長」を軸に、AIと世界観をつなぐBotやシステムを開発。  
Redis・LoRA・RAG・GPTを統合し、**人格成長型の対話AI**を構築しています。

---

## 🧠 注力中プロジェクト

### 📌 [YggPersonaBot（ユグペルソナボット）](https://github.com/sikanoumi/YggPersonaBot)  
> 感情・記憶・人格を持つキャラクターAIと成長型チャットBotを統合  
> Redis×FAISS×LoRA×GPT構成。Web UIで表情変化と感情分析も可。

- FastAPI / GPT-4 / Redis / FAISS / LoRA（SDXL）
- 感情表現に応じて表情切り替え・LoRA再学習
- Notion連携によるRAG検索・キャラ知識共有
- 感情分析 × 対話ログ抽出 → 再学習

### 🤖 [Persona Line Bot](https://github.com/sikanoumi/persona-line-bot)  
> LINEで動作するキャラクターBot。自然文による人格切替や記憶保存が可能。  
> Redisにユーザーごとの人格・状態・感情を管理し、成長型Botに応用。

## 💻 技術スタック

| 分野 | 使用技術 |
|------|----------|
| サーバー | FastAPI / uvicorn |
| AI/LLM | OpenAI GPT-4 / llama.cpp（予定） |
| 類似検索 | FAISS + SentenceTransformers |
| データ保存 | Redis（状態 / 感情 / 記憶） |
| 生成系 | LoRA + kohya_ss / SDXL / ControlNet |
| フロント | HTML / JS / Chart.js |
| ナレッジ | Notion API連携 |
